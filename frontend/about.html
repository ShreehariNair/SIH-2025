About the Initiative

Bridging Traditional Wisdom with Modern Healthcare

Project Overview

The AYUSH Dual Coding System is a pioneering initiative designed to bridge the gap between traditional Indian medicine (AYUSH) and modern healthcare classification systems. This system facilitates interoperability between the National AYUSH Morbidity and Standardized Terminology Electronic (NAMASTE) and the International Classification of Diseases 11th Revision (ICD-11), enabling seamless data exchange and analysis across diverse healthcare platforms.

Importance of Dual Coding

Dual coding with NAMASTE and ICD-11 is crucial for a comprehensive understanding of patient health by integrating traditional and modern medical perspectives. This enhances diagnostic accuracy, supports global data standardization, and promotes collaboration across medical systems.

Role of AYUSH

By incorporating AYUSH terminologies (Ayurveda, Yoga, Naturopathy, Unani, Siddha, Homeopathy) into a globally recognized coding system, we preserve its rich heritage and expand its impact and accessibility in mainstream healthcare practices.

Technology Stack

Our system is built on a robust MERN stack (MongoDB, Express.js, Node.js). MongoDB serves as our primary database for dual-coded data. The frontend, built with HTML and Tailwind CSS, offers a user-friendly interface, while APIs ensure smooth data communication between the frontend and backend. We've enhanced performance with optimized queries and added real-time features using WebSockets.

SIH Problem Statement

This project addresses the critical need for a standardized system to integrate traditional Indian medicine (AYUSH) with modern healthcare. The lack of interoperability between AYUSH terminologies and international classification systems like ICD-11 hinders data exchange, research, and comprehensive patient care. Our solution aims to bridge this gap by creating a dual coding system that accurately represents AYUSH concepts within a globally recognized framework.
